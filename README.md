# English Vocabulary Game

An interactive console game to practice English word translation.  
Includes levels, hints, and ranking for a more engaging experience.

---

## Features
- Multi-level vocabulary quizzes  
- Hints: word length, first letter  
- Score and ranking system  
- Replay option  
- End-of-game statistics  

---

## Tech Stack
- **Language:** Python 3  
- **Concepts:** loops, functions, dictionaries, user input validation  

---

## How It Works
1. Choose your level (easy, medium, hard).  
2. Translate words and get hints when stuck.  
3. See your total score and rank at the end.  

Example:

Level: Medium
Translate: "яблоко"
Hint: a_ _ _ e
Your answer: apple 
Score: 4/5 | Rank: Advanced
